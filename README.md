# Bond Calculator

Калькулятор для анализа покупки облигаций с полным расчётом доходности, окупаемости и налоговых последствий.

## Возможности

- **Поиск облигаций** — по ISIN или названию через API Московской биржи
- **Расчёт ключевых метрик:**
  - Чистая и грязная цена
  - Количество облигаций для заданной суммы инвестиций
  - Купонные выплаты (за период, годовой, общий до погашения)
  - Переплата за облигацию
  - Текущая купонная доходность
  - Доходность к погашению (YTM)
  - Эффективная доходность с реинвестированием
  - Чистая прибыль после налогов
- **Налоговый учёт:**
  - Стандартный счёт (13%)
  - ИИС тип А (вычет на взносы)
  - ИИС тип Б (освобождение дохода)
  - Комиссия брокера
- **Дополнительные функции:**
  - Сравнение с депозитом
  - Симулятор флоатера (what-if)
  - Расчёт дюрации, Z-Spread, G-Spread
  - Графики купонных выплат и амортизации
  - Экспорт в CSV и TXT
  - Сохранение состояния в LocalStorage
- **Поддержка валют** — RUB, USD, EUR, CNY, CHF, GBP с конвертацией

## Использование

1. Откройте `bond-calculator-v1.html` в браузере
2. Введите ISIN или название облигации для поиска
3. Нажмите «Загрузить данные об облигации»
4. При необходимости измените параметры вручную
5. Результаты рассчитываются автоматически

## Требования

- Современный браузер с поддержкой JavaScript
- Доступ к интернету для загрузки данных с API Московской биржи

## Технологии

- Чистый HTML/CSS/JavaScript
- Chart.js для графиков
- API Московской биржи (MOEX)
