# Bond Calculator

Калькулятор для анализа покупки облигаций.

## Возможности

- Расчёт доходности, окупаемости переплаты и налоговых последствий
- Загрузка данных облигаций через API Московской биржи
- Поддержка валютных облигаций
- Сравнение с депозитом
- Симулятор флоатера
- Адаптивный дизайн для мобильных устройств

## Использование

Откройте `bond-calculator-v1.html` в браузере.
